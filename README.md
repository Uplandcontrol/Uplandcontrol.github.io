# 🌱 슬기로운 방제생활

콩 · 들깨 · 참깨 · 녹두 재배 농가를 위한 **병해충 방제정보 조회 웹앱**입니다.
설치 없이 웹페이지 하나로, 병해충을 고르면 우리 창고에 있는 재고 중 방제 가능한 농약과
혼용 가능 조합을 바로 확인할 수 있습니다.

**바로 써보기 (GitHub Pages)**: `https://<사용자명>.github.io/<저장소명>/`

---

## ✨ 주요 기능

- **작물 선택**: 콩 · 녹두 · 참깨 · 들깨
- **병해충별 방제 농약 조회**: 정부 농약정보시스템 등록 정보(4,512건) 기본 내장
- **재고 엑셀 업로드**: 내가 가진 농약 재고와 자동 매칭, 재고 수량·보관장소 표시
- **작용기작 표시**: 코드별로 색상 구분 (같은 계열 파악에 도움)
- **인축독성 / 안전사용시기 / 안전사용횟수** 강조 표시
- **혼용정보 조회**: 두 농약을 같이 섞어도 되는지 확인
- **두 병해충 동시 발생 조합 찾기**: 병해충 2개를 고르면, 재고 있으면서 서로 혼용 가능한 조합만 추천
- **농약 이름 검색**: 작물 구분 없이 바로 검색

---

## 🚀 사용 방법

1. 위 GitHub Pages 링크로 접속 (또는 `index.html`을 다운받아 더블클릭)
2. **별도 업로드 없이 바로 사용 가능합니다** — 병해충/농약 등록 정보와 혼용정보(18,000여 건)가 이미 앱 안에 내장되어 있습니다
3. 여기에 본인 재고를 반영하고 싶다면, 화면 위쪽 "재고 엑셀 업로드"에서 본인 농약 재고 엑셀을 올리세요
   - 양식이 필요하면 앱 안에서 "재고입력 양식 다운로드"로 받을 수 있습니다
4. (선택) 더 최신이거나 본인만의 혼용정보가 있다면 "혼용정보 엑셀 업로드"로 올리면 내장 데이터 대신 그 내용이 적용됩니다
5. 작물 → 병해충 선택 → 방제 가능한 재고 농약 확인

---

## 🔒 데이터는 어디에 저장되나요?

- 업로드한 **재고·혼용정보 엑셀은 서버로 전송되지 않고, 접속한 브라우저의 로컬 저장소에만 저장**됩니다.
- 즉 내 재고는 나만 보고, 다른 방문자에게 공유되지 않습니다.
- 반대로, 다른 기기/브라우저로 접속하면 데이터가 안 보이므로 다시 업로드해야 합니다.
- 인터넷 연결 없이도(파일을 다운받아 두었다면) 로컬에서 실행 가능하지만, React/엑셀 처리 라이브러리를 CDN에서 불러오므로 **최초 로딩 시에는 인터넷 연결이 필요**합니다.

## ⚠️ 면책 조항

- 이 도구가 제공하는 병해충·농약 등록 정보는 **정부 농약정보시스템 공개 데이터**를 기반으로 하며,
  최신 등록 현황과 다를 수 있습니다. 실제 사용 전 반드시 **농약 라벨(포장지) 및 최신 등록 정보**를 확인하세요.
- **혼용정보는 각 제조사가 공개한 자료 또는 사용자가 직접 수집한 참고자료**이며,
  모든 조합·모든 농도·모든 지역 환경을 보증하지 않습니다.
  실제 혼용 전에는 소량 테스트 및 제조사·농촌지도소 확인을 권장합니다.
- 본 도구의 사용으로 발생하는 결과에 대해 제작자는 책임지지 않습니다.
